1.ให้หาว่า attribute ของ tag ต่อไปนี้มี อะไรบ้างและใช้งานอย่างไร

  • < h > style สี ขนาด ตัวหนา เอียง เส้นใต้ ฯ
  • < p > style ย่อหน้า หัวเรื่อง เป็นตัวแบ่งเนื้อหาในแต่ละย่อหน้านั้นๆ
  • < div > style ความสามารถ คือ การนำมาใช้แบ่งส่วนต่างๆ ของข้อมูลออกจากกัน
       คุณสมบัติ คือ ทำให้ข้อมูลนั้นถูกจัดวางเป็นส่วนๆ
       ลักษณะการนำไปใช้งาน คือ เป็น Block Element และ Content Model เป็น Flow
  • < img > มี 3 attribute src , alt, style
        src กำหนด url ของรูปภาพ
        alt ระบุข้อความสำรองสำหรับรูปภาพเมื่อไม่สามารถแสดงรูปภาพได้
        style ระบุสไตล์ CSS แบบอินไลน์สำหรับ element
  • < a > มี 3 attribute href, target, style
